# Tracheostomy Tube Dimensions — Reference Guide

Clinical reference tool for tracheostomy tube dimensions, cross-brand comparison, and alternative tube finder.

**Live tool:** https://scutts318.github.io/tracheostomy_dimensions_reference

## Contents
- OD, ID, length, and bend angle for all major manufacturers (Portex, Tracoe, Shiley, Bivona)
- Cross-brand size comparison at ISO sizes 7, 8, and 9
- Interactive alternative finder — select tube in situ and clinical goal to find ranked alternatives
- Data quality ratings per manufacturer with source citations

## Data sources
- Portex BLU/BLUselect: tracheostomyteam.org sizing chart (May 2024)
- Tracoe Twist: Atos Medical technical data sheet MC3043-TcEN 202306
- Tracoe Twist Plus REF 312: Atos Medical product datasheet (April 2023)
- Shiley Flexible: tracheostomyteam.org + Medtronic product specifications
- Bivona: tracheostomyteam.org (verify against ICU Medical IFU before clinical use)

## Clinical use
This tool is for clinical decision support only. Always verify dimensions against current manufacturer IFU before use. Final tube selection requires clinical assessment.
